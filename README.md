# Hi, I'm Aldo Larios

Software Engineer focused on building useful systems, not only writing code.

I like to understand the real business need first, design the right architecture, and then build solutions that work well in daily operations.

- Based in Mexico
- Spanish (native) and English (B1 — TOEFL)
- Open to remote work and relocation

## About Me

I have been writing code since 2019, and my biggest professional growth came from working at a tech consultancy.

There I learned that software development is not only about coding. It is also about understanding the problem, planning the structure, and delivering a product that is truly useful for the people who use it.

## What I Do

- System analysis and architecture
- Full-stack web development
- API and database design
- Software integration and automation

## Featured Projects

- **Loyalty Vault** — Loyalty platform with 25+ REST endpoints and custom admin panel.
- **Arcade** — Web system for game rental management, live time tracking, and point of sale.
- **Hestia** — Internal production monitoring system with automated calculations and reports.
- **InscribeME+** — Enrollment prototype with RFID and real-time notifications.
